* 目前此自動選課系統只支援 windows 作業系統，瀏覽器只支援 chrome 和 firefox 。

* (重要) 程式目前是開源的，大家皆可以使用。(學術用途)

* (重要) 如果正在使用此程式上選課系統，請不要用其他裝置再上去選課系統選課，兩邊可能會互相搶登入權限。(學校選課頁面有說明)

* (極重要) 啟用此程式前，請至 Windows設定 --> 系統 --> 顯示器 調整 「變更文字、應用程式與其他項目」為 「100%(建議選項)」
  。其他 Windows 版同樣也調整類似設定至100%。

* (重要) 開始本程式前，請先將 WebDriver 資料夾內想使用的瀏覽器丟到與 .jar(.exe) 同一個路徑。要使用 chromedriver 的話，要求您的
  chrome 版本為 v79~。
 
* (極重要) 如果有勾選可視化，在 START 之後想要關閉網頁一律使用「STOP」，請不要直接使用滑鼠點「X」關閉開啟的網頁，driver資源會無法釋放。

* (重要) 關閉程式的時候，請按程式右上方的「X」(絕對不要亂用工作管理員)，按「是」表示會儲存您的MAIL、LINE、課程，
  下一次開啟時就不需要重新輸入。(如果換學年，課程會要你輸入新的，但MAIL、LINE都會留著)
  (STOP 後有時網頁不會正常關閉，此時用以上「正當方式」關閉程式網頁就會正常關閉了)

* (重要) 按了 START 後，請確保程式經過完整一輪的進入選課系統、監控或選課、結束選課的流程，最後程式有重新回到選課
  初始頁面。如果 START 後沒有完整跑一輪，START 左邊的字會有紅色的提示，例如輸入錯學年度，要求您重開等等，也可以從
  紅色提示看到上一次網頁崩潰的原因(勾選重連瀏覽器才會在崩潰後重連，沒勾的話任何未預期的不完整網頁崩潰後就不會再開啟，建議一定要打勾)

* 程式開啟後，如果您使用了可視化模式，「請不要對程式開啟的瀏覽器頁面做多餘的按鈕、點擊、調解析度等動作」，本模式
  只用來觀賞及確認使用。(您還是可以做別的事情，它會自行在頁面裡面執行程式；它不能最小化，但可以壓在別的視窗下)

* 本程式可以跨學期、學年使用，主要是用來「撿課」使用，也就是當你可以「選」課的時候就可以用(第一階段不能使用，
  由於人數是無上限、而且是加選後抽籤，此階段只能手動自己加入)，優先資格抽籤的時候選課系統沒有開，所以也就無法
  使用。再次強調是撿課!

* 由於學校課程在選課前會不停地做更新，此程式是上網抓取學校每學期課程，請盡量在離選課系統較近的日子開程式抓取資
  料，如果過早抓取資料，進去程式後您點選「新增」的課程可能無法加入，此時請到 resourses 資料夾裡，把當前年度的
  XXXXCoursesData.json 刪除，讓程式自動幫你重新載就可以了。

* 如果您要用此程式拚搶課(2、3階段的時候)，您可以先提早開啟本程式，載好當前學年度課程資料 (依照您的網速跟效能，
  時間約 3 - 20 分鐘不等)，並加入課程以及設定好程式其他設定後，點選 START 讓程式開啟(選擇自動選課才會選課喔，且
  推薦要勾重連瀏覽器，因為一堆人搶著要進入選課系統，網頁時常壞掉需要重開)，此程式可在選課系統開放前就開始等待 ，
  它監測到選課系統開放後就會立即進入選課(如果此時過多人，學校網站整個掛了，其實程式一樣就是放著，他還是會嘗試連
  學校選課系統，只要他一修復，程式一定馬上就會是優先進入的。如果課早就被第二階段前選滿了就等撿課吧!每學期好課
  都一定會有人最後幾天丟課(我就是這樣撿的))。

* 於程式加入的課程最好要是您可以選的(您可以加入已經在選課系統中選上的課)，請不要加入一些外系不開放的課程、或是
  此階段不開放的課程，這類課程程式遇到會在每次登入選課系統後嘗試選課，可能會對學校網站造成過多的負擔，進而提升
  風險。而其他選課問題程式都會幫您檢查(例如加入重複時段的課程、加入的課程有些與線上選課系統重疊等)，請安心使用。

* 程式經過多次測試以及人性化設計，等同真人選課的方式(請盡量選擇推薦或以上的設定)，完善提供有用之相關功能，效能
  及所需硬體設備都以大眾能接受、使用的為主，且不會對學校網站造成負擔，可當成是一個常常查看選課系統的人(程式)。

* 程式執行中最穩定且使用最少資源的運行方式為：勾選使用 chrome 以及勾選可視化(看的到比較安心)。不勾選可視化只是
  瀏覽器看不到，但程式還在後台運作，不過前提是要懂程式上每個按鈕、選擇的影響才推薦使用。

* 驗證碼之4位平均準確度分別都約93%，真實情況近乎「每三次就有一次會登入」，也就是平均情況，事實上大部分都是一兩次。
